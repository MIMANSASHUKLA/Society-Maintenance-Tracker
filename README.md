# Society Maintenance Tracker

A web application for managing residential society operations, including resident management, maintenance tracking, complaints, notices, and payments.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- SQLite

## Features

- Resident Management
- Maintenance Tracking
- Complaint Management
- Notice Board
- Payment Records
- Secure Authentication

## Run Locally

```bash
npm install
npm run dev
```
